Customer Segmentation in Online Retail

Project Overview
This project focuses on segmenting customers based on their purchasing behavior using data from an online retail store. The goal is to analyze customer patterns and identify key segments that businesses can target for better marketing and sales strategies.

Dataset
- The dataset is an **Excel file (`Online Retail.xlsx`)**, containing transactional records from an online retail store.
- It includes details like **Invoice Number, Stock Code, Description, Quantity, Invoice Date, Price, Customer ID, and Country**.

Libraries Used
- `numpy`
- `pandas`
- `matplotlib`
- `seaborn`
- `warnings`

Key Steps in Analysis
- **Data Cleaning**: Handling missing values and incorrect data types.
- **Exploratory Data Analysis (EDA)**: Visualizing customer purchasing behavior.

Results & Insights
- The analysis provides insights into different types of customer behavior.
- It helps businesses optimize marketing campaigns based on customer segments.
